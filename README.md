## 🧠 How to Use

1. Open a word page on Cambridge Dictionary
   https://dictionary.cambridge.org

2. Click the extension icon

3. The extension will:
    - Scan the page
    - Display available audio files

4. Use:
    - ▶️ to preview
    - ⬇️ to download
